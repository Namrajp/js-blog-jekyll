# Shortcuts

- All Commands Pallette - cmd + shft + p
- Files - cmd + p
- Toggle terminal - ctrl + `(backtick)
- Toggle sidebar - cmd + b
- Comment selected - cmd + /
- Indent selected - cmd + } or {
- Select multiple lines hold alt and select
- Select similar tags - alt + d
